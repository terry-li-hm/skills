# Eudaimonia — The Operating System

> *A map of what a well-functioning mind does. Three layers, one goal: functioning well.*

Full philosophical context: [[Cognitive Operations Library]] (vault note).

## When to Consult What

### Inner — "am I in a state to think well?"

| Signal | Operation | Status |
|--------|-----------|--------|
| Reacting without noticing | **Self-awareness** | Not yet mined |
| Emotionally hijacked, can't think clearly | **Self-regulation** | Not yet mined |
| Repeating the same mistake | **Learning** | Not yet mined |

### Cognitive — "what kind of problem is this?"

| Signal | Operation | Skill |
|--------|-----------|-------|
| Don't know enough to act | **Truth-seeking** | `zetetike` |
| Need to structure future action | **Planning** | `bouleusis` |
| Need to assess quality or fitness | **Evaluation** | `kritike` |
| "Does it actually work?" | **Experimentation** | `peirasmos` |
| Too complex, too much | **Simplification** | `parsimonia` |
| Something broke, find the cause | **Debugging** | `diagnosis` |
| Need a rule of thumb | **Heuristics** | `praecepta` |
| All options feel obvious or stale | **Creativity** | `poiesis` |
| Side effects, "and then what?" | **Systems thinking** | `systema` |

### Inter — "how do I engage with others?"

| Signal | Operation | Status |
|--------|-----------|--------|
| Need to understand what someone thinks/feels/needs | **Empathy** | Not yet mined |
| Need to make myself understood | **Communication** | Not yet mined |
| Need to transfer work effectively | **Delegation** | `mandatum` |

### Meta — "which of the above?"

| Signal | Operation | Skill |
|--------|-----------|-------|
| The checklist ran out, rules don't fully apply | **Judgment** | `phronesis` |
| "What's right?" not just "what's wise?" | **Ethics** | Not yet mined |

## The Hierarchy

```
Values → what matters (not a skill — a commitment)
  ↓
Operations → how to pursue what matters well (this map)
  ↓
Domain skills → specific capabilities in specific areas
```

## Key Principles

- **Not a checklist.** A compass. Glance at it, then do the work.
- **Judgment sits above all.** It decides when to follow vs override any operation.
- **Separate generation from evaluation.** Never create and judge simultaneously.
- **"And then what?" × 3.** The simplest systems thinking tool.
- **"Am I learning or confirming?"** The sufficiency check from zetetike.
- **"What would change my mind?"** The debiasing check from phronesis.
