# Morning Review

Daily briefing to start the day with focus.

## Triggers

- "morning"
- "good morning"
- "gm"

## Steps

1. **Calculate today's date and day of week** (anchor: Jan 1, 2026 = Thursday)

2. **Check for overnight results** (if queued previous night):
   - Look for `Overnight Queue - [yesterday].md` in vault
   - Check for result files: `Job Alert Review`, `Pipeline Scan`, `Interview Prep`
   - Surface key findings and action items from overnight work

3. **Check job hunting status**
   - Read `/Users/terry/notes/Job Hunting.md`
   - Note any pending applications, follow-ups due, or waiting items

4. **Check for scheduled events**
   - Look for interviews, calls, or deadlines in the pipeline
   - Check if any daily notes mention upcoming commitments

5. **Review yesterday's daily note** (if exists)
   - Path: `/Users/terry/notes/Daily Notes/YYYY-MM-DD.md`
   - Look for incomplete items or carryover tasks

6. **Deliver a concise briefing**
   - Today's date and day of week
   - Overnight results summary (if any)
   - Top priorities / focus areas
   - Any time-sensitive actions
   - Keep it short â€” a quick summary, not a wall of text

## Output Format

```
**Tuesday, January 20, 2026**

Overnight results: [if queued]
- Job alerts: X new roles, Y worth reviewing
- Pipeline: Z stale items need follow-up
- [Key action items from overnight work]

Focus today:
- [Priority 1]
- [Priority 2]

Time-sensitive:
- [Any deadlines or scheduled items]

Pending from yesterday:
- [Carryover items if any]
```
